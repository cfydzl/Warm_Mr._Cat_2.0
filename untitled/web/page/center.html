外卖说明
1.  本餐厅位于福建省-福州市-晋安区-东三环路999号-福建工程学院（鳝溪校区） 2.  本餐厅对于福州三环内顾客下单提供免费配送服务，三环外顾客一律不以接单 3.  晋安区，台江区配送时间为40分钟，鼓楼区，仓山区配送时间为60分钟 4.  凡是上下班车流量高峰期，配送时间浮动为半小时 5.  顾客可根据配送员信息进行配送单状态查询 6.  以上一切解释权，最终归本店所有
订单说明
1.  顾客可以在当前的网站导航栏中点击菜单信息查看当前的餐厅菜单信息并加入购物车进行下单，该页面会显示当 前餐厅的各种不同的分类菜单，分别有店家推荐，主食，时素，时荤，浓汤，小食，酒水，顾客可以点击加减号对订单 进行修改，顾客选择完之后就可以下单后前往个人订单进行结算 2.  结算时候请选择是堂食还是外卖，若是堂食请进行堂食订桌，若无订桌前往餐厅就餐时本餐厅不保证顾客就餐时 期一定有空位，若已订桌，本餐厅会自动匹配用户的订单，进行堂食桌位的安排，在就餐时期可以之间前往就餐，无需 等待,若为外卖下单，本餐厅会安排派送员进行配送，个人订单也会相应显示配送状态与配送员电话 3.  凡是消费者下单成功，若想退单，必须联系商家，商家会根据当前订单的完成情况进行部分订单的退款，退单时 会收取10%的手续费 4.  以上一切解释权，最终归本店所有
堂食说明
1.  顾客可以在当前的网站导航栏中点击堂食信息查看当前的堂食订桌信息，该页面会显示当前餐厅各个时段下的四 种不同桌位，分别是两人桌，四人桌，六人桌，以及大包厢，大包厢可容纳8-12人，若需要前往本餐厅就餐，可提前在 本网站预定今天时间内的座位，每次就餐时间段默认为两小时，若当无餐厅座位时，可以选择外卖业务 2.  凡是消费者订桌成功，在结算订单之后，距离就餐时间大于两小时退订可以返回100%押金，距离就餐时间小于 两小时大于一小时退订可以返回100%押金，距离就餐时间小于一小时退订可以返回50%押金，订桌价格均为60，用餐 后可退还 3.  抵用券为堂食专属，不适用于外卖业务，每个账号每天只限购一种抵用券，在本网站页面支付之后请保存支付凭 证，前往本店就餐结算时可以出示使用，购买生日抵用券者须出示身份证或个人生日证明相关的证件 4.  在国家法定节假日，凡在本餐厅就餐，均享受九折优惠，不包括外卖且抵用券不能与节假日优惠同时使用 5.  以上一切解释权，最终归本店所有
